# Etch-A-Sketch

A small etch-a-sketch app I made using some JavaScript

### To Install

```
git clone https://github.com/Ben-893/Etch-A-Sketch
```

### To Use

```
Open the etch-a-sketch.html file in the browser, and draw something with the arrow keys!
```

### Technologies used

- JavaScript
- HTML
- CSS
